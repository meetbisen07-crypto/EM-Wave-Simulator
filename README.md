## 🌟 Project Overview

This project presents a comprehensive, visual approach to understanding electromagnetic (EM) waves. Rather than relying on static diagrams, it provides live, interactive simulations that allow students to manipulate wave parameters and observe the resulting physical behaviour in real time.

All simulations are grounded in the theoretical framework established by **James Clerk Maxwell's equations (1864)**, which unified the laws of electricity, magnetism, and optics — and predicted the existence of electromagnetic waves traveling at the speed of light. 💡

Simply open the link above in any modern browser to explore the simulation. No installation or setup is required. ✅
